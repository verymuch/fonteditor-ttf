fonteditor-ttf
==============

fonteditor中ttf模块的nodejs版本

提供ttf、woff、eot、svg格式字体的读取、生成和转换

提供ttf字形调整相关函数

提供otf转ttf相关函数

提供svg转字形相关函数

提供图片转字形相关函数

提供glyph轮廓相关函数

**此版本lib目录里的代码为 `在线编辑器核心(fonteditor-core)` 导出代码，不在此处编辑代码**

在线编辑器: [fonteditor](http://font.baidu.com)
